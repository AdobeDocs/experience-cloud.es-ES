# Audiencia en indicadores y grupos de características {#audience-overview}

Los despliegues de experiencias proporcionan segmentación de audiencia para indicadores de funcionalidades y grupos de funcionalidades. Puede combinar variables de contexto para controlar con precisión qué usuarios reciben una función.

Para el direccionamiento basado en el contexto (por ejemplo, segmentar según el idioma activo o el tipo de dispositivo del usuario), vea [Usar el contexto en las reglas de audiencia](using-context-in-audience-rules.md).
